# python

## while

#### Simple while
```python
a = 0

while a<10:
    print(a)
    a = a + 1
```

#### While with break
```python
a = 1

while True:
    print (a)
    a = a + 1
    if a>10:
        break
```

## list / array

**Remove an entry from list**
```python
fruitlist = ["appel", "aardbei", "banaan", "framboos", "kers", "banaan"]

print(fruitlist)

fruitlist.pop(0)

print(fruitlist)
```
__Result__
```text
['appel', 'aardbei', 'banaan', 'framboos', 'kers', 'banaan']
['aardbei', 'banaan', 'framboos', 'kers', 'banaan']
```
